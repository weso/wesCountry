Chart Help

ULR format

HOST/charts/CHART/[CONDITION_1]/[CONDITION_2]/[CONDITION_3][?OPTIONS]

Examples

  • HOST/charts/bar/YEAR(2010)
  • HOST/charts/bar/YEAR(2010)/COUNTRY(ESP,FRA,BRA,EGY)
  • HOST/charts/bar/YEAR(2010)/COUNTRY(ESP,FRA,BRA,EGY)/INDICATOR(ITU G)
  • HOST/charts/line/YEAR(2010)/COUNTRY(ESP,FRA,BRA,EGY)/INDICATOR(ITU G)?width=900&height=500
  • HOST/charts/pie/YEAR(2010)/COUNTRY(ESP)/INDICATOR(ITU A)
CHART values
  • bar Bar chart
  • line Line chart
  • pie Pie chart
  • polar Polar chart

NOTE
URLs MUST be encoded.
So the following URL:
HOST/charts/bar/YEAR(2010)/COUNTRY(ESP,FRA,BRA,EGY)/INDICATOR(ITU G)
Must be encoded as:
HOST/charts/bar/YEAR(2010)/COUNTRY(ESP,FRA,BRA,EGY)/INDICATOR(ITU%20G)

Default options

  • width Number. Sets graph width in pixels

    i.e. width=500

  • height Number. Sets graph height in pixels
  • margins Array of numbers. Sets graph margins.

    i.e. margins=[1, 1, 1, 1] set margins in order [top, right, bottom, left] to value 1% of the graph width, for left and right margins, and height, for top and bottom margins.

  • backgroundColour String representing a color.

    i.e. "none", "#666666", "#333", "red"

    backgroundColor="#333"

  • serieColours Array of strings representing colours. Colours for each serie.

    i.e. serieColours=["#333", "#444"]

  • groupMargin Number. Percentage of space in between of bar groups. Only applicable to bar charts.
  • barMargin Number. Percentage of space in between of bars. Only applicable to bar charts.

Axis options

The following options are valid for X and Y axes. Each option must be preceded by either "xAxis" or "yAxis".
  • title String. Sets axis name.

    i.e. xAxis.title = "Years"

  • colour String representing a colour. Sets the axis border colour.
  • margin Number. It's the margin between the chart and the axis.
  • tickColour String representing a colour. Colour for each tick line.
  • font-family String. Font family for axis labels.

    i.e. xAxis.font-family="Helvetica".

  • font-colour String representing a colour. Sets axis font colour.
  • font-size String representing a size. Sets axis font size.

    i.e. xAxis.font-size="16px"

  • values Array of strings. Only applicable for X axis with discrete values.

    i.e. xAxis.values=["January", "February", "March"]

Legend options

The following options must be preceded by "legend".
  • show Boolean. Indicates whether to show the legend.

    i.e. legend.show=false.

  • itemSize Number. Size of each legend colour bubble. It is measured in total graph width percentage.
  • font-family String. Font family for legend labels.

    i.e. legend.font-family="Helvetica".

  • font-colour String representing a colour. Sets legend font colour.
  • font-size String representing a size. Sets legend font size.

    i.e. legend.font-size="16px"

Series

Series are set as an array of objects. Each object has two values, name of the serie and values, an array of values.

i.e. series = [{ name: "First", values: [-1, 4, 5, 3, 6] }, { name: "Second", values: [1, 7, 5, 6, 4] }, { name: "Third", values: [0, 1, 2, 3, 4] }]
